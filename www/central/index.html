Artipie central

It's Artipie central repository manager. This is a list of public repositories:

  • central.artipie.com/maven - Maven repository
  • central.artipie.com/npmjs - npmjs mirror
  • central.artipie.com/bin - binary artifact storage

Maven repository

To use Artipie maven repository add these settings to your pom.xml file or configure via settings.xml: 

    
<project>
  [...]
  <pluginRepositories>
    <pluginRepository>
      <id>artipie</id>
      <name>artipie plugins</name>
      <url>http://central.artipie.com/maven</url>
    </pluginRepository>
  </pluginRepositories>
  <repositories>
    <repository>
      <id>artipie</id>
      <name>artipie builds</name>
      <url>http://central.artipie.com/maven</url>
    </repository>
  </repositories>
</project>

NPM proxy

NPM proxy for registry.npmjs.org may be useful for caching artifacts. Even if somebody removes its artifact from npmjs, Artipie will keep this dependency in cache. To download a package from Artipie use: npm install --registry=http://central.artipie.com/npmjs <package name> or set it as default repository: npm set registry http://central.artipie.com/npmjs

Binary artifact storage

Binary storage used by C or C++ developers to share binary libraries. To download library send GET request to http://central.artipie.com/bin/<library-name>.
